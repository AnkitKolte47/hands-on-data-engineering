# Hands-On Data Engineering

> Hands-on data engineering projects built in Python — covering ETL pipelines, multi-format parsing, OOP design patterns, and core Python concepts used in real-world data engineering roles.

---

## About This Repository

This repository is a collection of practical data engineering projects built from scratch in Python. Each project solves a real-world data problem and is designed to demonstrate job-ready skills — not just syntax knowledge.

Every project here covers concepts that appear in actual data engineering interviews and day-to-day work at companies of all sizes. The projects are ordered by complexity, so if you are a fresher, start from Project 01 and work your way forward.

## How to Use This Repository

Each project lives in its own folder and is completely self-contained. Every project folder has:

- A `README.md` with the full problem statement, day-wise breakdown, and step-by-step instructions

To get started with any project:

```bash
git clone https://github.com/your-username/hands-on-data-engineering.git
cd hands-on-data-engineering/01-shipment-etl-pipeline
```
